# 로그인 기능을 만들어본 프로젝트.

이 프로젝트는 제가 직접 백엔드, 프론트를 구현해 만든 로그인 프로젝트입니다. 세션쿠키를 기반으로 로그인의 상태관리를 하는 재밌는 코드랍니다. \
이전까지 제가 올렸던 코드와 비슷하게 나중에 제가 요긴하게 쓰려고 만들었답니다.
